Linux дистрибутивы в TOP-е

Debian GNU/Linux

Debian — операционная система, состоящая из свободного программного обеспечения с открытым исходным кодом, один из самых популярных, а также исторически и стратегически важных дистрибутивов GNU/Linux. Конечно, существуют проекты на основе других ядер - GNU Hurd, ядро FreeBSD, ядро NetBSD. Создан Яном Мёрдоком. Не препятствует установке проприетарного программного обеспечения. Есть бинарные блобы - немного. Конкретно нацелен на использование утилит GNU. Система крайне портируема и доступна на ~73 языках. Используется голосование для выбора лидера. Разрабатывается по сети независимо многими разработчиками. Использует пакетный менеджер dpkg c front-end-ом и программой для работы с репозиториями APT. Сейчас спонсируется SPI.

Debian - один из старейших и наиболее "влиятельных" дистрибутивов. The Debian Project был впервые анонсирован в 1993. Первый стабильный релиз появился в 1996.

Из-за такого статуса было принято решение разрабатывать систему по стандартам проекта GNU - открыто и свободно. Это привлекло внимание Фонда свободного программного обеспечения (FSF), который год спонсировал проект (1994-1995).

Что касается спонсирования, как уже было сказано, сейчас проект спонсируется некоммерческой организацией Software in the Public Interest - SPI (которую лидер проекта в то время Брюс Перенс создал в 1997). Эта организация частично спонсирует сейчас и другие проекты по разработке свободного программного обеспечения - так, например, что примечательно, дистрибутива Arch Linux, обсуждаемого на этом сайте, также проектов LibreOffice, FFmpeg и многих других.

Debian стал основой целого ряда (семейства) дистрибутивов, многие из которых крайне популярны или важны для решения специальных задач в определённой области. Это такие дистрибутивы, как: gNewSense (дистрибутив, одобренный FSF, с полностью свободным программным обеспечением - ни единого бинарного блоба), Kali Linux (удивительный дистрибутив, где есть всё для пентеста - дистрибутив для хакеров в популяризированном, но неправильном значении слова - см. Wikipedia), Knoppix, Linux Mint (основан на Ubuntu), Maemo, MEPIS, Musix GNU+Linux, Raspbian (Debian для Raspberry Pi), SteamOS (дистрибутив для Steam-а), TAILS (live CD дистрибутив для анонимности, приватности), Ubuntu (крайне популярный дистрибутив с медленной графической оболочкой Unity; разрабатывается Canonicial Ltd.™).

Произносится название дистрибутива "дэбиан". Это смешение имён создателя дистрибутива (Яна - Ian) и его жены на тот момент (Дебры - Debra).

Debian (как и большое количество других) использует классическую модель обновления релизами - определённые версии отдельных компонентов системы тестируют вместе, и когда результат выглядит стабильным создают новый стабильный релиз (поэтому основные компоненты системы могут быть несколько устаревшими).

Графическая оболочка по умолчанию - GNOME 3, также специально поддерживаются KDE, Xfce and LXDE. Конечно, можно установить и любую другую.

Относительно менеджмента пакетов, в отличие от других семейств систем GNU/Linux, менеджмент в семействе Debian осуществляется с использованием двух отдельных компонентов: dpkg и APT.

dpkg используется для установки, удаления и предоставления информации о пакетах - файлах с расширением .deb, которые представляют из себя архивы ar c 2 архивами .tar (и ещё одним файлом) внутри (с информацией и самой программой). Здесь следует подчеркнуть, что большинство программ для GNU/Linux собраны именно в этом формате, если Вам нужны бинарные файлы, и нацелено на эту операционную систему или (ещё уже) на Ubuntu.

Для работы с репозиториями, зависимостями, используется Advanced Packaging Tool, - фактически, более высокоуровневая надстройка над dpkg с расширенным функционалом. Программа автоматизирует получение, конфигурацию и установку пакетов. Была написана по проекту начатому Браяном Вайтом для Debian. О самих репозиториях Debian следует сказать, что официальные являются самыми общирными среди всех дистрибутивов (более 50000 пакетов).

Хотя Debian имеет репозиторий с ПО с закрытым исходным кодом, но всё же это система с открытым исходным кодом и она очень важна для становления свободных систем и систем с открытым исходным кодом будущего и определения самого понятия свободного ПО и ПО с открытым исходным кодом. The Debian Project создал три крайне значимых документа для всего GNU/Linux, open-source и libre сообщества: Конституция Проекта Дебиан, Критерии Debian по определению свободного ПО и Социальный договор Debian.

Надо сказать, что определение свободного программного обеспечения определяется Debian не так как Free Software Foundation. Критерии Debian по определению свободного ПО определяют ПО и другие компоненты ОС с открытым исходным кодом.

Очевидно, что у Debian большое сообщество, которое его поддерживает.
